# File Organizer Script

Python automation script designed to organize files automatically by extension and category.

The project focuses on:

* Automation
* File handling
* Error handling
* Workflow optimization
* Structured debugging

## Features

* Automatically sorts files
* Creates folders dynamically
* Handles duplicate names
* Supports multiple file types
* Reduces manual organization work

## Technologies

* Python
* OS module
* shutil module

## Example Categories

* Images
* Documents
* Videos
* Archives
* Music

## Objectives

This project was built to improve practical Python automation skills and file system manipulation knowledge.
